Vad är ram minne?

Ram minne, även kallat endast ram, är den del i datorn som fungerar som korttids minne. Det är det minne som används för att ha igång de program du använder. Till skillnad från SSD minne kan du inte lagra saker på ditt RAM minne.

Viktigt att tänka på:


• Generation: Det är viktigt att du väljer ram från den generationen som ditt moderkort är gjort för så att dem är kompatibla med varandra
• Mängd: Du måste se till att du väljer så mycket ram så att datorn alldrig får slut på ram. Om du får slut på ram kan det göra att du inte kan köra alla program du vill och att datorns hastighet saktas ner. Ett bra sätt att kolla så att du får tillräckligt med ram är att söka upp tester där du kan se hur mycket som går åt för just det du vill göra.
