This website is used for getting the times for the sunrise and sunset on specific coordinates. To use the website, press the point at which you want to see the times on the map and you will see the times on the right. Alternatively you can manually type in the coordinates in the text fields below the map and then hit the "Check Time" button

JavaScript must be enabled for the website to work!
